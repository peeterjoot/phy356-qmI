\makeproblem{Trace invariance for unitary transformation (2008 PHY355H1F final 1b.)}{problem:qmIexamPractice2008Dec:1}{
Show that the trace of an operator is invariant under unitary transforms, i.e. if \(A' = U^\dagger A U\), where \(U\) is a unitary operator, prove \(\tr(A') = \tr(A)\).

\index{trace!invariance}
\index{unitary}
} % problem

\makeanswer{problem:qmIexamPractice2008Dec:1}{

\index{commutation}
The bulk of this question is really to show that commutation of operators leaves the trace invariant (unless this is assumed).  To show that we start with the definition of the trace

\begin{equation}\label{eqn:qmIexamPractice2008Dec:570}
\begin{aligned}
\tr(AB)
&= \sum_n \bra{n} A B \ket{n} \\
&= \sum_{n m} \bra{n} A \ket{m} \bra{m} B \ket{n} \\
&= \sum_{n m}
\bra{m} B \ket{n}
\bra{n} A \ket{m}
\\
&= \sum_{m} \bra{m} B A \ket{m}.
\end{aligned}
\end{equation}

Thus we have
\begin{equation}\label{eqn:qmIexamPractice2008Dec:1b:10}
\tr(A B) = \tr( B A ).
\end{equation}

For the unitarily transformed operator we have
\begin{equation}\label{eqn:qmIexamPractice2008Dec:590}
\begin{aligned}
\tr(A')
&= \tr( U^\dagger A U ) \\
&= \tr( U^\dagger (A U) ) \\
&= \tr( (A U) U^\dagger ) \\
&= \tr( A (U U^\dagger) ) \\
&= \tr( A ) \qedmarker
\end{aligned}
\end{equation}
} % answer
